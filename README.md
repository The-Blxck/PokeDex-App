## Features

- Search for Pokémon by name
- Displays Pokémon **image**, **name**, and **type(s)**
- Colored type badges for better visualization
- Responsive **grid layout**
- Hover effect on cards for interactivity
- Error handling for invalid Pokémon names
- Works with **Enter key** or Search button

---

## Technologies Used

- **HTML5** – Structure of the page
- **CSS3** – Styling the Pokédex and cards
- **JavaScript (ES6)** – Fetching Pokémon data and rendering cards dynamically
- **PokéAPI** – Free public API for Pokémon data
